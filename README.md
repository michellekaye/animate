## Animate

Animate is a playground for me to practice CSS animations.

Visit [animate.michelle-kaye.com](http://animate.michelle-kaye.com)

## Getting Started

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies

- Next.js
- React
- Typescript
- SCSS
- Github actions
- AWS S3
